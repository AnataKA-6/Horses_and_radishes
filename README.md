# Horses_and_radishes

#========================================================================================#
Проект компьютерной версия настольной игры "Лошади и Редиски" (ОС Windows 10/11).

#========================================================================================#
Цель игры - стать самым богатым фермером за 60 дней.

В распоряжении фермера есть:
стойла с лошадьми, тренировки лошадей, поля для взращивания редисок на продажу, лотереи, скачки, аукционы, задания от заказчиков.

#========================================================================================#
Реализовано на Python 3.14.5 с помощью нейросети Deepseek (https://chat.deepseek.com/).

Анимация создана с помощью komiko.app.

Звуки игры взяты с zvukipro.com.

Изображения созданы с помощью myneuralnetworks.ru и chatgpt.com.

Музыка создана с помощью suno.com.

Озвучивание создано с помощью topmediai.com.

#========================================================================================#

Необходимые библиотеки для работы с кодом:
1) Pygame-ce (Community Edition) для воспроизведения звуков и музыки

   (cmd: py -m pip install pygame-ce);
   
2) Pillow v12.2.0 для работы с изображениями 

   (cmd: py -m pip install pillow);
   
3) PyInstaller v6.20.0 для создания исполняемого файла

   (cmd: py -m pip install pyinstaller).

#========================================================================================#

Структура проекта:
   horse_board_game/
├── horse_board_game.py          # Основной файл игры
├── horse_images/                 # Изображения лошадей
├── gif/                          # Анимации
├── sounds/                       # Звуки (опционально)
├── saves/                        # Папка для сохранений
│   └── (создаётся автоматически)
└── icon.ico                      # Иконка приложения


build_exe.bat - файл генерации исполняемого exe-файла в папке dist (предварительно очищает содержимое данной папки).
